Installation

You can install Taslonic in your project either using a Package Manager or getting it directly from a CDN.

Package Manager

If you manage your project dependencies using a package manager (strongly recommended), you can install Taslonic using either NPM or Yarn:

npm install --save @glorious/taslonic yarn add @glorious/taslonic

CDN

If you're not into package management or want to give Taslonic a quick try using a standalone HTML file, you can load it from a third-party CDN provider. There are two options: JS Deliver and UNPKG.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@glorious/taslonic/taslonic.css"> <!-- Load one of the bundles below according the engine (react or vue) you are using in your project --> <script src="https://cdn.jsdelivr.net/npm/@glorious/taslonic/react/index.js"></script> <script src="https://cdn.jsdelivr.net/npm/@glorious/taslonic/vue/index.js"></script> <link rel="stylesheet" href="https://unpkg.com/@glorious/taslonic/taslonic.css"> <!-- Load one of the bundles below according the engine (react or vue) you are using in your project --> <script src="https://unpkg.com/@glorious/taslonic/react/index.js"></script> <script src="https://unpkg.com/@glorious/taslonic/vue/index.js"></script>

You're ready!

Once the installation is complete, you're ready to start using Taslonic components.
